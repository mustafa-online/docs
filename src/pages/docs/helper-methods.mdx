## Display a image

By default Shopper provide a helper to load image on the current uploads disk. You can easely change disk on your `shopper/system.php` config file.

```php
// To display image available on the avatars disk, you set the second parameter to avatars.

shopper_asset($user->avatar_location, 'avatars')
```

## Website Currency

If you want to display correct money format, Shopper provide a simple helper to help you, not based on default `money_format` php function :

```php
shopper_money_format($product->price_amount)
```
